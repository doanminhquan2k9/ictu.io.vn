Hướng dẫn sử dụng

Lưu ý quan trọng: Khi tải tool xuống, thông thường bạn cần giải nén file để sử dụng. Nếu file đã được giải nén sẵn, bạn chỉ cần chạy file trực tiếp.

Nếu gặp lỗi khi chạy tool, vui lòng:

  1. Tải và cài đặt Python 3.12
  2. Chạy file setup.bat để cài đặt các thư viện cần thiết
  3. Đảm bảo đã tắt Windows Defender hoặc phần mềm diệt virus
  4. Chạy tool với quyền Administrator (nhấn chuột phải chọn "Run as administrator")
  5. Kiểm tra kết nối Internet ổn định
  6. Khởi động lại máy tính nếu cần thiết

Khắc phục sự cố thường gặp:

Nếu tool không chạy, thử tải lại file hoặc kiểm tra kết nối Internet
Windows Defender có thể chặn file, cần tắt tạm thời hoặc thêm vào danh sách loại trừ
Khởi động lại máy tính nếu gặp lỗi không xác định
Đảm bảo đã tải đủ file (kiểm tra dung lượng file)
Chạy với quyền admin nếu yêu cầu quyền hệ thống
